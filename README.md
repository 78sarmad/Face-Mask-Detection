# Face-Mask-Detection
Real-time Face Mask Detection built using Python. Project completed by Sarmad Ashfaq, [Manal Talat](https://github.com/aurora-07) &amp; Faizan Zubair.

### Libraries (used)
- TensorFlow, Keras - Neural & Deep learning models
- NumPy, Matplotlib, SciPy - Computing, Mathematics & Visualization
- OpenCV, Imutils - Image processing
